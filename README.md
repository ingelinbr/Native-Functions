Hva er FastNotes

FastNotes er en enkel notatapp utviklet i React Native med Expo.
Appen lar brukeren opprette, lagre og lese personlige notater.

Funksjonalitet i appen:
- Vise en liste over alle notater
- Opprette nye notater med tittel og innhold
- Se detaljer for et valgt notat
- Navigere tilbake til hovedskjermen
- Notater lagres midlertidig i appens state og blir ikke lagret permanent.

Krav for å kjøre prosjektet

Følgende må være installert:
- Node.js
- npm
- Expo CLI
- Android Studio (for emulator) eller Expo Go på mobil

Kjøre appen på Android studio:
- Start Android Studio
- Åpne Device Manager
- Start en Android emulator
- Når emulatoren kjører, trykk "a" i terminalen eller velg Run on Android device i Expo.
